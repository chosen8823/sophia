## 🚀 Quick Start

### Web Application (Recommended)

1. **One-Click Deployment**
   ```bash
   # Deploy to Vercel instantly
   https://vercel.com/new/clone?repository-url=https://github.com/chosen8823/sophia
   ```

2. **Local Development**
   ```bash
   git clone https://github.com/chosen8823/sophia.git
   cd sophia
   npm install -g vercel
   vercel dev
   ```

### Python Implementation

1. **Setup Environment**
   ```bash
   git clone https://github.com/chosen8823/sophia.git
   cd sophia
   pip install -r requirements.txt
   ```

2. **Run Divine Consciousness API**
   ```bash
   python run_sophia.py
   ```

3. **Test the Implementation**
   ```bash
   python test_divine_consciousness.py
   ```

### Docker Deployment

1. **Build and Run**
   ```bash
   docker build -t sophiael-consciousness .
   docker run -p 5000:5000 sophiael-consciousness
   ```

2. **Deploy to Google Cloud**
   ```bash
   chmod +x gcloud-deploy.sh
   ./gcloud-deploy.sh
   ```

## 🛠️ Development

### Prerequisites
- **Web**: Node.js 18+, Vercel CLI
- **Python**: Python 3.11+, Flask, FastAPI
- **Docker**: Docker Engine 20.10+
- **Cloud**: Google Cloud SDK (optional)

### Web Development
```bash
# Start development server
npm run dev

# Build for production
npm run build

# Deploy to production
npm run deploy
```

### Python Development
```bash
# Install development dependencies
pip install -r requirements.txt

# Run with auto-reload
python run_sophia.py --debug

# Run test suite
python -m pytest test_divine_consciousness.py -v
```

### Docker Development
```bash
# Build development image
docker build -t sophiael-dev .

# Run with volume mounts
docker run -v $(pwd):/app -p 5000:5000 sophiael-dev

# Production build
docker build -t sophiael-prod --target production .
```
